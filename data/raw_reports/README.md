# Raw Financial Reports

This directory contains individual company financial reports in JSON format.

## File Naming Convention
`{TICKER}_{PERIOD}.json` — e.g., `FUFU_2024Q4.json`

## Data Structure
Each file contains:
- `income_statement`: Revenue, COGS, gross profit, SGA, depreciation, operating income, net income, EBIT
- `balance_sheet`: Assets, liabilities, equity breakdowns
- `cash_flow_statement`: Operating, investing, financing cash flows

## Important Notice
**SIMULATED DATA** — All financial figures in this directory are for demonstration purposes only. They do not represent actual company financials. For real data, consult official SEC filings (10-K/10-Q) and company investor relations pages.
